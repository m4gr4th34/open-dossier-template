A spiral galaxy, generated from a seed

The same runtime that drives the deterministic n-body orrery here drives statistical structure: a four-arm log-spiral, a bulge and a halo, all generated from integer seeds — not stored. Rendered by figures/galaxy.js on the Phase 1 primitives in figures/figures.js.

Drag to pan, scroll to zoom, or jump to a region. The zoom spans roughly ten orders of magnitude — from the whole galaxy down toward the Solar System. The orrery proved deterministic orbits; this proves the runtime extends to procedural structure, with an honest "you are here" that is correctly one point at galaxy scale.

A standalone demo, not one of the frozen editions. It shows the same runtime crossing regimes — from the deterministic n-body orrery to statistical structure — with a ~10-order zoom and an honest sub-pixel "you are here". JavaScript off falls back to the sealed static poster; JavaScript on upgrades it to this live figure.
