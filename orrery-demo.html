The zoom-orrery, alive through the vendored runtime

A standalone demo — not one of the frozen editions. The figure below is rendered by figures/orrery.js, which composes the Phase 1 primitives in figures/figures.js.

Drag to pan, scroll to zoom, or jump to a region. The play-speed couples to the zoom scale (scaleAwareTime): zoomed in, time slows so the inner planets stay legible instead of blurring; zoomed out, time speeds up so the giants visibly move. The two belts and the Oort cloud are generated from integer seeds (seededScatter), not stored.

A standalone demo, not one of the frozen editions — it exists to show the vendored runtime driving the real figure, drag, zoom, and play included. Because it is not a frozen edition, the freeze script-src rewire never applies here. With JavaScript off it falls back to the sealed static poster baked into the page; with JavaScript on, that floor upgrades to this live interaction.
